How to Use

  1. First Use: Click the bookmarklet and enter your Hugging Face token (free!)
  2. Highlight Text: Press Ctrl + Shift and drag over text
  3. Get Insights: An overlay appears with a librarian-style summary
  4. Ask Follow-ups: Type questions in the text box
  5. Save Conversations: Click "Save Dialogue" to download as text
  6. Toggle On/Off: Click the bookmarklet again or use Ctrl + Alt + S

Features

  • Text Analysis: Get concise summaries of any text on the web
  • Conversational Interface: Ask follow-up questions about highlighted content
  • Export: Save your research conversations as text files
  • Privacy-focused: Your API token stays in your browser
  • Works Everywhere: Compatible with most websites
  • Image Support: Experimental support for highlighting images (OCR)

Troubleshooting

  • Token Issues: Clear localStorage with localStorage.clear() in console (F12)
  • CORS Errors: Some sites block external scripts - try using on simpler pages
  • Model Errors: The tool will automatically try alternative models if one fails
  • Image Capture: Still experimental - works best with text for now

SlimScreen is open source and under active development
